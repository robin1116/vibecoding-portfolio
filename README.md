# html-css-js-demo

Vite 기반의 간단한 정적 웹 프로젝트입니다.

## 요구사항
- Node.js 18+
- npm 9+

## 설치
`ash
npm install
`

## 개발 서버 실행
`ash
npm run dev
`
Vite 개발 서버가 시작되며, 기본 주소는 http://localhost:5173 입니다.

## 프로덕션 빌드
`ash
npm run build
`

## 빌드 결과 미리보기
`ash
npm run preview
`

## 프로젝트 구조
`
index.html
main.js
style.css
`

## 배포 메모
- dist/ 디렉토리의 정적 파일을 원하는 정적 호스팅에 업로드합니다.
